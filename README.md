# Sales Data Analytics Project Task-2

## Project Overview

This project demonstrates a complete data analytics workflow using Microsoft Excel, MySQL, and Power BI. The objective was to analyze sales performance, customer behavior, profitability, and regional trends through data cleaning, querying, visualization, and business reporting.

## Project Workflow

### 1. Data Preparation and Cleaning (Excel)

* Removed duplicates and handled missing values
* Standardized data formats
* Created calculated columns such as Profit Margin and Average Order Value
* Applied IF, SUMIFS, and COUNTIFS functions
* Performed advanced analysis using Pivot Tables and Pivot Charts

### 2. Database Management and Analysis (MySQL)

* Created relational database tables for Orders and Customers
* Imported CSV datasets into MySQL
* Performed JOIN operations between tables
* Executed business queries for revenue, profit, customer, and regional analysis

### 3. Business Intelligence Dashboard (Power BI)

* Built interactive KPI cards for Total Sales, Total Profit, Profit Margin, and Total Customers
* Created visualizations including:

  * Sales by Region
  * Profit by Category
  * Monthly Sales Trends
  * Top Customers Analysis
  * Customer Segment Distribution
* Added interactive slicers and filters

## Key Business Insights

* Identified the highest revenue-generating region
* Determined the most profitable product category
* Evaluated the impact of discounts on profitability
* Analyzed customer segment performance
* Discovered seasonal sales patterns

## Tools Used

* Microsoft Excel
* MySQL
* Power BI

## Outcome

The project demonstrates practical skills in data cleaning, SQL querying, business analytics, dashboard development, and insight generation for data-driven decision-making.
